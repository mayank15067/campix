# 🚀 Campix – Smart Campus Complaint & Event Management System

## 📌 Overview

Campix is a web-based platform designed to streamline campus issue reporting and event management.
Students can report complaints, upvote issues, and explore campus events, while admins manage both complaints and events efficiently.

---

## 🎯 Problem Statement

In many colleges:

* Complaints are ignored or delayed
* No proper system exists to track issues
* Event information is scattered and not centralized

---

## 💡 Solution

Campix provides a unified platform where:

* Students can raise complaints and vote on them
* Events can be listed and viewed easily
* Admin can monitor complaints and events from a single dashboard

## 🔑 Features

### 📢 Complaint Management

* 📝 Submit complaints
* 👍 Upvote important issues
* 📌 Track complaint status

### 🎉 Event Management (NEW 🚀)

* 📅 Students can **create/list events**
* 👀 Users can **view all events**
* 🛠️ Admin can **access and monitor events**

### 👨‍💼 Admin Panel

* View all complaints
* Manage complaint status
* View all listed events
